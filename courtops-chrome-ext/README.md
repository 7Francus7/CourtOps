# CourtOps Quick Actions

Extensión de Chrome para acceso rápido al dashboard y detección de contactos.

## Funcionalidades

- **6 accesos rápidos** al dashboard (Reservas, Clientes, Caja, Check-in, Config)
- **Escáner de contactos**: detecta teléfonos, emails e Instagram en cualquier página web
- **Alta rápida**: botón "+ Agregar" abre `/clientes/nuevo` con los datos pre-llenados
- **Menú contextual**: seleccioná texto en cualquier web y elegí "Agregar a CourtOps como cliente"
- **Indicador de estado**: punto verde si la instancia está online

## Instalación (2 minutos)

1. Abrí Chrome → `chrome://extensions/`
2. Activá **Modo desarrollador** (esquina superior derecha)
3. Clic en **Cargar extensión sin empaquetar**
4. Seleccioná la carpeta `courtops-chrome-ext/`
5. El ícono aparece en la barra de herramientas

## Configuración

1. Clic en el ícono de la extensión
2. En "URL de CourtOps" ingresá tu URL (ej: `https://app.courtops.net`)
3. Guardá — los accesos rápidos apuntarán a tu instancia

## Uso del escáner

1. Navegá a la web de un club de pádel prospecto
2. Abrí la extensión → "Escanear"
3. Se detectan teléfonos, emails e Instagram automáticamente
4. Clic en "+ Agregar" para ir directo al formulario de alta en CourtOps
